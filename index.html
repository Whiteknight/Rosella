Rosella by Whiteknight

A collection of tools and building blocks for the Parrot Virtual Machine.

Rosella is a library of patterns and practices for use with the Parrot Virtual Machine. The goal of Rosella is to provide a library of building blocks for programmers on Parrot. By having these tools and components available in a portable, usable way, programmers should be empowered to create high quality, maintainable software faster.

Dependencies

  • Parrot
  • Winxed

Install

                winxed setup.winxed build
                winxed setup.winxed test
                winxed setup.wixned install
